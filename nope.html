$ # Yaad rakho in chezon ka syntax:

1. Abstract class banane ke liye:
   → from abc import ABC, abstractmethod
   → class naam(ABC): likh ke @abstractmethod decorator lagao

2. Private variable:
   → self.__variablename  (double underscore)

3. Inheritance:
   → class Child(Parent):
   → super().__init__(params) — parent ka __init__ call karo

4. Override:
   → child class mein same method ka naam — apna code likhao

5. Static variable:
   → class ke andar, __init__ ke bahar
   → ClassName.variable se access

6. Global variable:
   → function ke andar global x likhao pehle

7. Recursion:
   → base case zaroori (if n==0 return)
   → warna infinite loop

$ # Common exam mistakes:
   → _init_ likhna instead of __init__ (do underscore!)
   → super() bhool jana
   → abstract method mein pass likhna bhool jana
   → private variable ek underscore _ se banana (galat)
