Sabse Zaroori Syntax
from abc import ABC, abstractmethod   # abstract ke liye
class MyClass(ABC):                    # ABC inherit karo
    @abstractmethod                      # decorator zaroori
    def method(self): pass

class Child(Parent):                   # inheritance
    def __init__(self, a, b):
        super().__init__(a, b)           # parent call

self.__variable = 0                    # private (__ double)
global x                               # function mein bahar ki var
Common Mistakes
✗  _init_   →  ✓  __init__   (do underscore dono taraf)
✗  _grade   →  ✓  __grade    (private ke liye 2 underscore)
✗  super(Parent).__init__()
           →  ✓  super().__init__()
✗  abstract method mein body
           →  ✓  sirf pass likhna hai
